# Controle Financeiro Pessoal

Projeto desenvolvido para portfólio acadêmico no curso de Análise e Desenvolvimento de Sistemas.

## Funcionalidades
- Cadastro de entradas e saídas
- Categorização de movimentações
- Cálculo automático de saldo
- Filtros por tipo
- Persistência com LocalStorage

## Tecnologias
- HTML5
- CSS3
- JavaScript (Vanilla)

## Objetivo
Demonstrar lógica de programação, manipulação de dados e regras de negócio aplicadas a um sistema real.
